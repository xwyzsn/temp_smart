## macos 用户

下载[文件地址](https://github.com/xwyzsn/temp_smart/actions/runs/21097194785/artifacts/5164575529)

下载后同样解压

右键 -> 打开 会弹出不能运行

此时点击系统设置->隐私与安全 -> 允许server运行 然后仍要打开即可


然后打开浏览器 输入[http://127.0.0.1:8000](http://127.0.0.1:8000)即可
